# Rabbika Azmi - Portfolio

Personal portfolio website for Rabbika Azmi, an engineer and researcher in making.

## About This Project

This is a React-based portfolio website featuring a unique 3D folding effect. The site showcases my work, research interests, and professional journey.

## Features

- 3D fold animation effect
- Responsive design
- Modern React implementation
- Easy to customize and deploy

## Setup & Installation

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Customization

### Update Personal Information

1. **Content**: Edit `src/Content.js` to add your bio, projects, research, and contact links
2. **Styling**: Modify `src/Content.module.css` to customize colors and layout
3. **Meta Tags**: Update `public/index.html` if needed for SEO
4. **Homepage URL**: Change the homepage URL in `package.json` to your GitHub Pages or custom domain

### Add Your Projects

Fill in the placeholders in `src/Content.js` with:
- Project titles and descriptions
- Links to live demos or repositories
- Research areas and interests
- Contact information and social media links

## Deployment

To deploy to GitHub Pages:

1. Update the `homepage` field in `package.json` with your GitHub Pages URL
2. Run `npm run deploy`

For custom domain deployment, add a `CNAME` file in the `public` folder with your domain name.

- React 16.13.1
